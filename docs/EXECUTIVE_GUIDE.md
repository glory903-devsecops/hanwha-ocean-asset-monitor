# 🏆 Executive Portfolio Guide: Hanwha Ocean AX Implementation
> **Target Audience: IT Division Head (CIO/CTO), Recruitment Manager (AX Team)**

### 1. "Why this matters?" (ROI Perspective)
In a traditional shipyard, IT assets and production IoT devices are managed separately. This fragmentation leads to:
- **Delayed Recovery**: IT doesn't know when a PLC fails until it's reported.
- **Inaccurate Inventory**: Manual Excel sheets lead to procurement errors.
- **Operational Gaps**: No single source of truth for the entire yard's digitial health.

**This solution provides a 360-degree view.** It is not just a dashboard; it is a **Centralized Command Center** that understands the relationship between a Server in the data center and a Robot on the assembly line.

### 2. Strategic Alignment (Smart Yard 2030)
- **Connected Yard**: Integrating MQTT (IoT) and SNMP (IT) under one hood.
- **Automated Yard**: RPA Hooks for auto-ticketing ensure "Zero-Manual-Effort" in procurement lifecycles.
- **Autonomous Yard**: The data foundation built here is ready for **CBM (Condition-Based Maintenance)** AI integration in Phase 2.
