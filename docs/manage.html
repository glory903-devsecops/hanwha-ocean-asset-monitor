📋 데이터 관리 및 아키텍처 정보

  • • 등록 경로 (POST /api/v1/assets): 새로운 자산이 등록되면 FastAPI가 Pydantic 모델을 통해 유효성을 검사합니다.
  • • 수정 경로 (PUT /api/v1/assets/{id}): 자산의 상태(Status)나 위치 정보 변경 시 SQLAlchemy를 통해 DB를 업데이트합니다.
  • • 삭제 경로 (DELETE /api/v1/assets/{id}): 자산 폐기 시 트랜잭션을 안전하게 처리하며 모든 내역은 감사 로그(Audit Trail)에 기록됩니다.
  • * 본 화면은 백엔드 API와의 인터랙션을 시각화하기 위해 설계된 관리 인터페이스 UI입니다.
